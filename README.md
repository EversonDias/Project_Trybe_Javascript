# Acesso ao projeto

Para acessar e executar este projeto, siga os passos abaixo:

1. Certifique-se de ter o Node.js e o npm instalados em sua máquina.

2. Clone o Projeto

```bash
git clone  ZooFunction
```

3. Entre no projeto

```bash
cd ZooFunction
```

4. Instale as Dependência

```bash
npm install
```

4. inicie o projeto

```bash
npm test
```
